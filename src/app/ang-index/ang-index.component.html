The Angular Index is full of resources for terminology specific to Angular. There are times where we get caught up with the actual work and forget the terms in which we are using to progress through our work. At least, that's been my experience. The Ang-Index is a tool that can be used to refresh the memory.

AOT (Ahead of Time)

Short Description - Before the browser downloads, HTML and TypeScript is compiled into efficient JavaScript code during the build process.

Bootstrapping

Short Description - The Root module ngModule or appModule gets it's properties from Bootstrapping which inserts components into the index.html file to initialize the application.

read more
Class Decorator

Short Description - Appears before a class definition, declares the class to be of a type, and provide metadata related to that class

  1. @Component()
  2. @Directive()
  3. @Pipe()
  4. @Injectable()
  5. @NgModule()
read more
Class Field Decorator

Short Description - Statement before a field class. Declares the type of field of the class. To name a few...

  1. @Input
  2. @Output
read more
Component

Short Description - A class with the @Component decorator that associates itself with a template(ie. html file) in order to create a view.

read more
Custom Element

Short Description - Allows you to create an HTML tag that is controlled by JavaScript code. Also called a web component, it is supported by most browsers and others browser by using polyfils.

read more
Data Binding

Short Description - The relationship between HTML and data. There are several types of binding all which rely on a users clicks, keystrokes and touches.

  1. Interpolation
  2. Property binding
  3. Event binding
  4. Attribute binding
  5. Class binding
  6. Style binding
  7. Two-way data binding with ngModel
read more
Dependency Injection (DI)

Short Description - A design pattern that communicates dependencies or services to other parts of the application that require them.

read more
Directive

Short Description - A class that modifies the DOM and it's elements.

  1. Components use @Component() (an extension of @Directive()) to associate a template with a class.
  2. Attribute directives modify behavior and appearance of page elements.
  3. Structural directives modify the structure of the DOM.
read more
Lazy Loading

Short Description - Speeds up the application load time by splitting the application into bundles. Unlike eager-loading, dependencies are loaded only as needed.

read more
Life Cycle Hook

Short Description - Allows you to link into the life cycle of dependencies and components when they are created, Updated and Destroyed.

  1. ngOnChanges: When an input/output binding value changes.
  2. ngOnInit: After the first ngOnChanges.
  3. ngDoCheck: Developer's custom change detection.
  4. ngAfterContentInit: After component content initialized.
  5. ngAfterContentChecked: After every check of component content.
  6. ngAfterViewInit: After a component's views are initialized.
  7. ngAfterViewChecked: After every check of a component's views.
  8. ngOnDestroy: Just before the directive is destroyed.
read more
